# Stock and Crypto Market Forecasting Project

This project is designed to analyze stocks and cryptocurrencies using technical indicators and machine learning concepts in order to generate a general market impression.

## Features
- Analyze stock and crypto market data
- Use technical indicators such as SMA, EMA, RSI, and MACD
- Generate a general impression:
  - Bullish
  - Bearish
  - Neutral
- Educational and analytical purpose only
